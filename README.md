# Simulador de Financiamento — Marco Túlio

Simulador de financiamento imobiliário (Caixa / Minha Casa Minha Vida) com geração de **link compartilhável** para corretores: o corretor coloca o WhatsApp dele e gera um link que o cliente usa pra simular sozinho, com o botão final caindo direto na conversa.

🔗 **Ao vivo:** https://simulador.marcotulio.pro

## Stack
Site estático — HTML/CSS/JS em arquivo único (`index.html`). Sem build, sem dependências.

## Deploy
Publicado no **Netlify** (deploy manual). Para atualizar: edite o `index.html` e faça novo deploy (arraste a pasta no painel do Netlify ou conecte este repositório para deploy automático).
